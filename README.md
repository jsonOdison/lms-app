## **Overview**
ElevateHub is a compact learning platform built for the assignment requiring a fast, AI-assisted LMS prototype.  
The focus is on **speed**, **AI-assisted development**, and a **clean, functional demo**.

---

## **Features**

### **1. Landing Page**
- Clear hero section  
- Primary CTA: **View Courses**

### **2. Courses Page**
- List of sample courses  
- Each course includes a **title** + **short description**

### **3. Course Detail Page**
- Overview section  
- Topic breakdown  
- **Progress indicator** via `localStorage`  
  - **Enrolled / Not Enrolled**  
- Enroll button  
- Reset progress confirmation modal

### **4. AI-Powered Course Tutor**
- Chatbot powered by **OpenAI**  
- Uses **course-specific content only**  
- Acts like a **RAG-lite tutor**  
- Answers only from the provided course material

### **5. Deployment**
- Fully deployed on **Vercel**  
- Public GitHub repository
